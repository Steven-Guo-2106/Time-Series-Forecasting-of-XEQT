# Time-Series-Forecasting-of-XEQT
This project investigates next-day stock price forecasting using historical market data of the iShares Core Equity ETF Portfolio.
Modelled daily open, high, low, close, and volume features within a time-series framework.
Utilized standardization and ridge linear regression to improve generalization and address multicollinearity.
Model performance is evaluated on an out-of-sample test set with no look-ahead bias.
